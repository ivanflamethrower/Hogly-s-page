План обучения фулстак джун

Старт (20.08.2021) – Финиш (дд.мм.гггг)

  1. Вёрстка. HTML и CSS. (3-4 месяца)
    • Курсы:
      Htmlacademy (HTML и CSS);
      WebReference (HTML и CSS + фреймворки Bootstrap и Font Awesome);
    • Научиться верстать макеты на .psd (Adobe Photoshop) и/или на Figma;
    • Научитсья работать с программами: Avocode, Browser Sync (и/или Gulp + Livereload); научиться применять
      SASS & Emmet; пользоваться Pix to pix (чтобы макеты не разъезжались); выбрать редактор кода (скорее
      всего уже будет выполнено к этому моменту);
    • Начать пользоваться W3C Валидаторами HTML и CSS кода (позволяет получать валидный код на выходе);
    • Изучить адаптивность;
    • Изучить фреймворки и библиотеки для фронтенд: Bootstrap, Foundation, Materialize CSS;
    • Изучить понятия и термины: Семантика, структура, форматирование, микроразметка, SPA, лендинг,
      контентная часть, отрисовка страницы, препроцессоры, сброс, флоаты, флексы, блочная вёрстка,
      позиционирование: разница между static, absolute и relative, понятие очистки float,
      Mobile First, Table of contents, методологии БЭМ и OOCSS, кроссбраузерность, кроссплатформенность,
      минификация, прафило 14 килобайт и т.д.;
  2. Программирование на JS. (soft: 1 месяц; skill: 4-6 месяца);
    • Курсы:
      Htmlacademy (более для новичков: 3 недели); learn.javascript.ru (более для скилловых: 3-4 месяца);
    • Изучить библиотеку jQuery;
    • В идеале научиться применять следующие библиотеки и решения (возможно усвоить за неделю):
      Slick, Owl Carousel, Masonry, jQuery UI Sortable, Magniffic Popup, Remodal, CuteSlider,
      Select2, Face Detection, Progress.js, Tooltipster, Typed JS, PrettyEmbed, fullpage.js,
      Velocity.js, Tree.js, Slideout.js, Focus Point, Liquid Images, Vide, Swipe, jQuery Autocomplete,
      JsBarcode, Input Mask, Chart.js, Vegas, Isotope, jQuery Vibrate, Image Cropper.;
    • Для написания SPA: React JS, Angular JS, Vue.js, Aurelia JS;
    • Научитсья работе с: WebPack, PUG, Font Ninja, Visual Event;
    • Уметь к этому моменту работать с: Git & Github;
    • Если углубляться в изучение данного языка: Bable JS, TypeScript, CoffeScript;
  3. Backend - разработка.
    Основные языки:
    Node.js (JavaScript) - универсален, можно с ним работать если углубленно учить JS;
    Python - мощный язык спонсируемый гуглом, не самый популярный в снг, дольше учить;
    PHP - востребован в СНГ, быстрый старт и выход на заработок;
    Ruby - новый, относительно недавно появившийся язык.
    • PHP (Junior: 3-5 месяцев)
      1. Курсы (выбрать между): https://htmlacademy.ru/ ; книга: PHP 7 в подлиннике;
      2. Необходимо познакомиться с популярными CMS-системами и фреймворками:
        Wordpress, Opencart, Mod X, YII2, Laravel;
      3. Адрес PHP библиотеки: packagist.org;
    • Python (Junior: 5-6 месяцев)
      1. Курсы: либо подобрать самим, либо найти книгу - Byte of Python.
        Для более продвинутого уровня: Learn Python the Hard Way;
      2. Изучить фреймворки: Django и Flask;
      3. Изучить библиотеку: Requests;
    • Node.js (JavaScript) (Junior: после изучения JS, хватит 1 месяца)
      1. Курсы:
        Официальная документация
        Htmlacademy (платный курс)
        Скринкаст от Ильи Кантера
      2. Изучить CMS-систему: Ghost CMS;
      3. Изучить фреймворк: Total JS;
      4. Изучить популярные библиотеки и фреймворки: Express, Socket.io, Meteor, Requests, Async, Underscore;
      5. Библиотека фреймворков: npmjs.com;
      6. Комъюнити разработчиков: pypi.python.org;
  4. Базы данных
    • Выбрать и изучить системы управления базами данных (СУБД):
      MongoDB, MySQL, PostgreSQL, SQL Server, Oracle, Firebase;
    • Топ выбор для PHP: MySQL, PostgreSQL
      Топ выбор для Node.js: MongoDB, Firebase;
